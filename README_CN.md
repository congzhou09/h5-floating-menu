### 使用

下载或通过npm（__npm install h5-floating-menu__）安装，
```

// 初始化，指定配置
window.H5FloatingMenuConfig = {};
require ('h5-floating-menu');
```
或者通过script标签引入，
```
<script>
// 初始化，指定配置
window.H5FloatingMenuConfig = {};
</script>

<script src="path/to/h5-floating-menu.min.js"></script>
```
